## 🎮 Project Overview

**Rock Paper Scissors Game** is a Python-based mini game where the user competes against the computer.

The program uses Python logic and the random module to generate the computer's choice and decide the winner based on standard Rock-Paper-Scissors rules.

## 🎲 Game Rules

| User                | Computer | Result |
| ------------------- | -------- | ------ |
| Rock                | Scissors | Win    |
| Scissors            | Rock     | Lose   |
| Paper               | Rock     | Win    |
| Rock/Paper/Scissors | Same     | Draw   |
